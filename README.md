#PWC Cyber Workshop 2026

A PwC cybersecurity workshop assignment focused on security log analysis and threat detection.

##  Description

This project is part of the PwC cybersecurity workshop program for 2026. It contains tools and scripts for analyzing security logs and identifying potentially rogue user access and suspicious activities within a system.

## Project Overview

The workshop assignment involves detecting and reporting security anomalies by:
- Analyzing system logs for suspicious patterns
- Identifying rogue IP access attempts
- Flagging suspicious login attempts
- Generating comprehensive security reports

## Repository Contents

### Scripts
- **checkLogs.py** - Python script for analyzing security logs and identifying suspicious activities

### Reports
- **Report.csv** - Security incident report containing detected anomalies
  - Lists usernames and reasons for flagging (RogueIPAccess, SuspiciousAttempt)
  - Contains findings from the security log analysis

## Technologies Used

- **Python** - Primary language for log analysis and threat detection
- **CSV** - Format for reporting findings
